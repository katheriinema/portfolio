Tech Stack 🚀

  • Back End
    • Godot & GDscript: Built backend with Godot's own programming language, GDscript. Godot software features were also leveraged to make game development more efficient
    • Supabase: Used to host our users' data, such as number of penguins owned, total revenue, days played, user login details, and more.
  • Front End
    • Godot: Assets hand drawn and imported and adjusted within Godot.
    • Vercel: Used as intermediate for Supabase and our front end in Godot to deploy.

Description

In this cozy simulation game, users manage a colony of penguins. This project challenged me both technically and creatively—designing mechanics in Godot, coding in GDScript, and crafting art assets. Activities include fishing, building structures, and collecting new penguin species.
